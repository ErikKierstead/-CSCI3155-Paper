- This code is a real world example of using the subprocess command to run a shell command and pull in standard input
- Requires NMAP and Python in Linux to run (you can sudo 'yum install nmap' or 'sudo apt-get install nmap' to install it)
- Runs like: python nmapParser.py <ip address or range of addresses>
     as in : python nmapParser.py 192.168.0-1.1-255     scans most home networks

- Originally written for the Research Computing Group to scan network devices.
